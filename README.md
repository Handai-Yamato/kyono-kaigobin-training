# 目次（Table of Contents）

dist ...成果物フォルダ。公開するデータはすべてここで管理
  ├─asset ...全体に共通するデータを格納
  │  ├─css ...scssをコンパイル後のcssを格納
  │  ├─img ...画像を格納
  │  └─js ...コンパイルしたものを格納
  └─project ...下層ページのhtmlを格納
  └─index.html ...トップページのhtml



sass ...scssファイルを格納
 ├─global
 │  ├─setting
 │  │   _color..............プロジェクト全体の色を管理します。
 │  │   _font...............プロジェクト全体のフォントを管理します。
 │  │   _z-index............プロジェクト全体のz-indexプロパティを管理します。
 │  └─mixin
 │      _bg-image...........`background-image`を指定するmixinです。
 │      _breakpoint.........メディアクエリ用のmixinです。
 │      _centering..........要素を中央揃えするmixinです。
 │      _pseudo.............疑似要素用のmixinです。
 │
 ├─foundation
 │   _reset................ブラウザ固有のcssを打ち消すスタイルです。
 │   _base.................要素セレクタと属性セレクタのデフォルトスタイルです。
 │
 ├─layout
 │   _header...............ヘッダーエリアのコンテナブロックのスタイルを指定します。
 │   _footer...............フッターエリアのコンテナブロックのスタイルを指定します。
 │   _main.................メインコンテンツエリアのコンテナブロックのスタイルを指定します。
 │   _sidebar..............サイドバーエリアのコンテナブロックのスタイルを指定します。
 │
 ├─component
 │   _heading..............見出しのベーススタイルです。
 │   _keyword..............キーワードBOXのベーススタイルです。
 │   _search-box...........検索ボックスのベーススタイルです。
 │
 ├─project
 │   _home.................homeページ固有のスタイルです。
 │
 └─utility
     _display...............`display`を指定するためのユーティリティクラスです。
     -margin................`margin`を指定するためのユーティリティクラスです。
