お届けについて

商品は佐川急便またはヤマト運輸でのお届けとなります。通常、ご注文確定メールから３〜８営業日でお届けいたします。
ご注文の混雑時など、お届け日が前後する場合がございます。

お荷物お届けまでの目安

  • ※一部地域、離島は除く。
  • ※上の図は目安の為、日数は前後する場合がございます。
  • ※沖縄・離島等への配送は、ヤマト運輸を利用する場合があります。
